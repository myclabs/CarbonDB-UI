About

CarbonDB is a life cycle assessment (LCA) open dataset mainly centered on greenhouse gas and energy impacts.

It is structured as a domain ontology in terms of the Ressource Description Framework (RDF) World Wide Web Consortium specification.

Source data is maintained by My C-Sense as an open source project on GitHub, from where it can be freely downloaded, consulted, and edited using any ontology editor, for instance protégé.

Changes on ontology can be followed here. Please feel free to contribute by submitting issues or directly submitting an improved version of the ontology.

This project benefited from research jointly carried on by My C-Sense and computer science laboratory LIRIS from Université de Lyon. Its main innovative feature is the fact that dependency relations among processes can be defined at a macro-level, which makes the model easier to understand and maintain.
